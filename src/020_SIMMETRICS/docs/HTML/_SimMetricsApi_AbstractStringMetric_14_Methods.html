AbstractStringMetric Methods
AbstractStringMetric | Legend
Methods
Method Description
BatchCompareSet does a batch comparison of the set of strings with the given comparator string returning an array of results equal in length to the size of the given set of strings to test.
BatchCompareSets does a batch comparison of one set of strings against another set of strings returning an array of results equal in length to the minimum size of the given sets of strings to test.
GetSimilarity gets the similarity measure of the metric for the given strings.
GetSimilarityExplained gets a div class xhtml similarity explaining the operation of the metric.
GetSimilarityTimingActual gets the actual time in milliseconds it takes to perform a similarity timing. This call takes as long as the similarity metric to perform so should not be done in normal circumstances.
GetSimilarityTimingEstimated gets the estimated time in milliseconds it takes to perform a similarity timing.
GetUnnormalisedSimilarity gets the un-normalised similarity measure of the metric for the given strings.
